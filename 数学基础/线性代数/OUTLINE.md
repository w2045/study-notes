# 线性代数 · 课程大纲

10 章，纯数学基础定位，穿插实例加深理解。

## 作业形式约定

- **编程题** = `notes.md` + `理论题/` + `编程题/`（homework.py + grader.py + solutions.md）
- **理论题** = `notes.md` + `理论题/`（homework.md + solutions.md，无编程题/ 子目录）
