# 概率论 · 课程大纲

7 章，为 ML/DL 中的概率建模与统计推断服务。

## 作业形式约定

- **编程题** = `理论题/` + `编程题/`
- **理论题** = `理论题/`（仅此子目录）

## 章节清单

| # | 章 | 作业形式 | 要点 |
|---|-----|----------|------|
| 01 | 概率公理与条件概率 | 理论题 | 样本空间、概率公理、条件概率、全概率公式 |
| 02 | 随机变量与分布函数 | 编程题 | PMF/PDF/CDF、采样模拟、分布可视化 |
| 03 | 期望、方差、矩 | 编程题 | 期望、方差、协方差、矩生成函数 |
| 04 | 常见离散分布 | 编程题 | Bernoulli、Binomial、Poisson、Geometric |
| 05 | 常见连续分布 | 编程题 | Uniform、Gaussian、Exponential、Beta |
| 06 | 贝叶斯定理与贝叶斯推断 | 编程题 | 先验/后验、共轭先验、Beta-Binomial |
| 07 | 最大似然估计 | 编程题 | 似然函数、对数似然、MLE 推导与数值求解 |
