## Q1

<details><summary>点击查看</summary>

`Cmd/Ctrl + Shift + P` → 输入关键词：
- `theme` → 选择 `Preferences: Color Theme`
- `json` → 选择 `Preferences: Open User Settings (JSON)`
- `zoom` → 选择 `View: Zoom In` 或 `Editor Font Zoom In`

命令面板的关键词匹配是模糊的，不需要输入完整命令名。
</details>

---

## Q2

<details><summary>点击查看</summary>

`Cmd/Ctrl + P` → 输入 `prog` 或 `PROGRESS` → 回车打开。

VS Code 的模糊匹配很智能——`prog` 就能匹配到 `PROGRESS.md`。
</details>

---

## Q3

<details><summary>点击查看</summary>

**Alt + Click**：按住 `Alt`（macOS 是 `Option`），在 3 个不同位置点击。出现 3 个闪烁光标后，开始打字——3 个位置同时出现相同内容。

**Cmd/Ctrl + D**：
1. 选中第一个 `pass`
2. 按 `Cmd/Ctrl + D` 3 次，选中所有 4 个 `pass`
3. 输入 `# TODO`——所有位置同步替换
</details>
