## 1. 为什么需要 Markdown？

写学习笔记时，你需要同时处理**普通文字**、**代码块**、**数学公式**、**表格**、**图片**。如果用 Word，排版会占用大量注意力；如果用纯文本，又无法表达结构。

Markdown 就是为此而生的——用极简的符号标记结构，最终渲染成美观的文档。你正在看的这份笔记就是用 Markdown 写的。

**核心理念**：写的时候专注于内容，排版交给工具。

## 2. 基本语法

### 2.1 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

`#` 的数量决定层级。绝大多数文档用 2-3 级就够。

### 2.2 强调

```markdown
**粗体**       →  **粗体**
*斜体*         →  *斜体*
~~删除线~~     →  ~~删除线~~
`行内代码`     →  `行内代码`
```

### 2.3 列表

无序列表用 `-` 或 `*`：

```markdown
- 第一项
- 第二项
  - 子项（缩进 2 格）
  - 另一个子项
```

有序列表用数字：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 2.4 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](path/to/image.png)
```

### 2.5 引用

```markdown
> 这是一段引用文字。
> 可以多行。
>
> 空一行开始新段落。
```

### 2.6 分隔线

```markdown
---
```

三个或更多 `-` 产生一条水平线。

## 3. 代码块

行内代码用单反引号：`` `print("hello")` `` → `print("hello")`

代码块用三个反引号包裹，标注语言获得语法高亮：

````markdown
```python
def greet(name):
    """向 name 打招呼。"""
    return f"Hello, {name}!"
```
````

效果：

```python
def greet(name):
    """向 name 打招呼。"""
    return f"Hello, {name}!"
```

常用语言标注：`python`, `cpp`, `javascript`, `bash`, `markdown`, `latex`, `json`, `yaml`。

## 4. 表格

```markdown
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| a      |  b   |      c |
| 长内容 |  短  | 中等   |
```

| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| a      |  b   |      c |
| 长内容 |  短  | 中等   |

- `:---` 左对齐，`:---:` 居中，`---:` 右对齐
- 竖线 `|` 不需要对齐——渲染后自动对齐

## 6. 数学公式（行内与块级）

Markdown Preview Enhanced 支持 LaTeX 数学公式嵌入：

```markdown
行内公式: $E = mc^2$

块级公式:
$$ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$
```

效果：行内公式 $E = mc^2$，块级：

$$ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$

> 我们会在下一章详细学习 LaTeX 公式语法。

## 9. VS Code 中的 Markdown

### 预览

打开 `.md` 文件后：

| 平台 | 侧边预览（推荐） | 独立标签页预览 |
|------|:---------------:|:-------------:|
| macOS | `Cmd + K V` | `Cmd + Shift + V` |
| Ubuntu | `Ctrl + K V` | `Ctrl + Shift + V` |
| Windows | `Ctrl + K V` | `Ctrl + Shift + V` |

- **侧边预览** (`K V`)：在编辑器旁边打开，同步滚动，适合边写边看
- **独立标签页** (`Shift + V`)：在新标签页打开，适合只看不改

> 如果安装了 Markdown Preview Enhanced 扩展，侧边预览还会额外支持数学公式渲染、流程图等高级功能。

### 常用快捷键

| 功能 | macOS | Ubuntu / Windows |
|------|-------|-------------------|
| 加粗 | `Cmd + B` | `Ctrl + B` |
| 斜体 | `Cmd + I` | `Ctrl + I` |
| 侧边预览 | `Cmd + K V` | `Ctrl + K V` |
| 独立标签页预览 | `Cmd + Shift + V` | `Ctrl + Shift + V` |
| 折叠区域 | `Cmd + Shift + [` | `Ctrl + Shift + [` |
| 展开区域 | `Cmd + Shift + ]` | `Ctrl + Shift + ]` |
| 命令面板 | `Cmd + Shift + P` | `Ctrl + Shift + P` |
| 快速打开文件 | `Cmd + P` | `Ctrl + P` |

> **记忆技巧**: macOS 用户按 `Cmd`，Linux/Windows 用户按 `Ctrl`，其余按键相同。

### 实时同步滚动

Markdown Preview Enhanced 预览窗会和编辑窗同步滚动。点击预览中的任意位置，编辑器会跳到对应位置。
