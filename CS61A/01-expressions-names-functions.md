## 1. 直觉引入：程序就是计算器 + 菜谱

Python 程序本质上只做两件事：

1. **计算**：像一个超级计算器，对数据执行运算
2. **定义**：给值和操作命名，像写菜谱一样组织逻辑

打开 Python 交互环境（终端输入 `python3`），你会看到 `>>>`。这就像一个对话——你输入一行代码，Python 计算并回答你。

```python
>>> 2 + 3
5
>>> 7 * 6
42
```

这和你用计算器没有本质区别。但 Python 的强大在于你可以给自己的计算过程「命名」：

```python
>>> 半径 = 5
>>> 面积 = 3.14159 * 半径 * 半径
>>> 面积
78.53975
```

`半径` 和 `面积` 就是名字（变量）。你不再需要每次重复输入数字——代码变成了可复用的配方。

## 2. 表达式与值

### 2.1 基本表达式

> **定义（表达式）**：一个**表达式**是 Python 可以求值的一段代码。求值后产生一个**值**。

最简单的表达式是字面量——直接把值写出来：

```python
>>> 42          # 整数字面量
42
>>> 3.14        # 浮点数字面量
3.14
>>> "你好"      # 字符串字面量
'你好'
>>> True        # 布尔字面量
True
```

Python 中 `#` 后面的内容（包括 `#` 本身）是**注释**，Python 会忽略它。注释写给人看。

### 2.2 调用表达式

> **定义（调用表达式）**：形式为 `操作符(操作数, ...)`。Python 先对操作数求值，再将值传入操作符（函数）。

```python
>>> max(3, 7, 2)    # max 是内置函数，返回最大值
7
>>> abs(-5)          # abs 返回绝对值
5
>>> pow(2, 10)       # pow(a, b) 返回 a 的 b 次方
1024
```

`max(3, 7, 2)` 的执行过程：
1. Python 识别这是一个调用表达式
2. 对操作数 `3`, `7`, `2` 求值 → 它们本身就是值
3. 将 `(3, 7, 2)` 传入函数 `max`
4. `max` 返回最大值 `7`

### 2.3 复合表达式

表达式可以嵌套，Python 按照**优先级**和**从左到右**的规则求值：

```python
>>> 2 + 3 * 4      # * 优先级高于 +，等价于 2 + (3 * 4)
14
>>> (2 + 3) * 4    # 括号改变优先级
20
>>> 10 / 3         # / 总是返回浮点数
3.3333333333333335
>>> 10 // 3        # // 是整数除法（向下取整）
3
>>> 10 % 3         # % 是取余数
1
>>> 2 ** 10        # ** 是乘方
1024
```

> **常见错误**：`10 / 3` 在 Python 3 中返回 `3.333...`，不是 `3`。如果你想要整数结果，用 `//`。

## 3. 名字与赋值

### 3.1 赋值语句

> **定义（赋值语句）**：`name = expression` 将右边表达式的值绑定到左边的名字。

```python
>>> x = 10          # 把值 10 绑定到名字 x
>>> x + 5           # 现在可以用 x 参与计算
15
>>> y = x * 2       # 右边先求值：10 * 2 = 20，再绑定给 y
>>> y
20
```

赋值是**语句**，不是表达式。语句不产生值——它执行一个动作（在内存中建立名字到值的绑定）。

### 3.2 变量与重新赋值

同一个名字可以先后绑定到不同的值：

```python
>>> a = 5
>>> a = a + 3    # 先算右边 a+3=8，再把 8 绑定给 a
>>> a
8
```

这需要精确理解求值顺序：**先求值右边，再绑定到左边**。所以 `a = a + 3` 中右边的 `a` 用的还是旧值 `5`。

### 3.3 命名规则

- 只能包含字母、数字、下划线 `_`
- 不能以数字开头
- 区分大小写：`name` 和 `Name` 是不同的名字
- 不能用 Python 关键字：`if`, `else`, `while`, `def`, `return` 等

```python
>>> 2bad = 1      # 错误！不能以数字开头
SyntaxError: invalid syntax
>>> if = 3        # 错误！if 是关键字
SyntaxError: invalid syntax
```

## 4. 定义函数

这是本章最重要的部分。函数让你把一段计算逻辑封装起来，给它一个名字，以后反复使用。

### 4.1 `def` 语句

```python
>>> def 平方(x):
...     return x * x
```

**解剖这个定义**：

| 部分 | 含义 |
|------|------|
| `def` | 关键字，告诉 Python「我要定义一个函数」 |
| `平方` | 函数名，你自己取 |
| `(x)` | 参数列表，`x` 是形式参数（parameter） |
| `:` | 函数头结束，接下来是函数体 |
| `return x * x` | 函数体，计算并返回结果 |

调用它：

```python
>>> 平方(5)
25
>>> 平方(3) + 平方(4)
25        # 9 + 16
```

### 4.2 参数与返回值

函数可以有多个参数，也可以没有参数：

```python
>>> def 加法(a, b):
...     return a + b
>>> 加法(3, 7)
10

>>> def 问候():
...     return "Hello!"
>>> 问候()
'Hello!'
```

如果没有 `return` 语句，函数返回 `None`（一个特殊值，表示「什么都没有」）：

```python
>>> def 打印并忽略(x):
...     print("收到:", x)   # print 只是显示在屏幕上
...                        # 没有 return
>>> 结果 = 打印并忽略(42)
收到: 42
>>> 结果                    # 没有返回，默认是 None
>>>                        # （Python 不显示 None）
```

### 4.3 文档字符串

函数内的第一行可以用三个引号写文档：

```python
>>> def 百分比(部分, 整体):
...     """计算 部分 占 整体 的百分比。"""
...     return (部分 / 整体) * 100
>>> help(百分比)     # 显示文档
百分比(部分, 整体)
    计算 部分 占 整体 的百分比。
```

## 5. 求值规则（环境模型入门）

理解 Python 如何执行代码，是避免大多数 bug 的关键。规则只有两条：

### 规则 1：表达式的求值

当你输入一个表达式，Python 从左到右计算它：
1. 字面量 → 直接得到值（`42` 就是 `42`）
2. 名字 → 在当前环境（一张名字→值的映射表）中查找
3. 调用表达式 → 先求值操作数和操作符，再执行调用

### 规则 2：函数调用的执行

当调用一个用户定义的函数时，Python 执行以下步骤：

1. **创建一个新的局部帧**（小而独立的名字→值表）
2. **将实际参数绑定到形式参数**（在局部帧中）
3. **在局部帧中执行函数体**
4. **遇到 `return` 时返回，或到达函数末尾返回 `None`**
5. **销毁局部帧**

**用一个例子走一遍**：

```python
>>> def f(x, y):
...     return x + y
>>> a = 3
>>> b = 5
>>> f(a, b)
8
```

执行过程：
1. `def f(x, y)` → 在全局帧中创建函数 `f`
2. `a = 3` → 全局帧中 `a` 指向 `3`
3. `b = 5` → 全局帧中 `b` 指向 `5`
4. `f(a, b)`：
   - 创建局部帧
   - 在局部帧中：`x` 绑定到 `a` 的值（3），`y` 绑定到 `b` 的值（5）
   - 在局部帧中执行 `return x + y` → 计算 `3 + 5 = 8`
   - 返回 `8`，销毁局部帧

> **关键直觉**：函数调用是「隔离的」。函数内部做的事情不会影响外面（除非使用可变对象，后面会讲）。

## 6. 断言（`assert`）

断言是一种快速测试的手段：你预期某件事为真，如果不是就报错。

```python
>>> 平方(5) == 25
True
>>> assert 平方(5) == 25    # 通过，没有任何输出
>>> assert 平方(0.5) == 0.25
>>> assert 平方(-3) == 9
```

断言是静默的——通过时不输出任何东西。如果断言失败：

```python
>>> assert 平方(5) == 26
Traceback (most recent call last):
  ...
AssertionError
```

把断言和 `def` 结合起来，你就有了一个带自动测试的函数：

```python
def 平方(x):
    """返回 x 的平方。"""
    assert type(x) in (int, float), "输入必须是数字"
    return x * x

# 自动测试
assert 平方(5) == 25
assert 平方(0) == 0
assert 平方(-4) == 16
print("✅ 所有测试通过")
```

## 7. `print` 理解：显示 vs 返回

这是一个初学者非常容易混淆的点：

| 概念 | 含义 | 例子 |
|------|------|------|
| `print(...)` | 在屏幕上显示内容 | `print("hello")` → 屏幕显示 `hello` |
| `return` | 函数把值交给调用者 | `return x` → 调用处得到 `x` 的值 |

```python
>>> def 加和打印(a, b):
...     print("计算中...")
...     return a + b
>>> 加和打印(2, 3)
计算中...        # ← print 的输出
5                # ← return 的值（Python 交互环境显示它）
```

`print` 是给人看的；`return` 是给程序用的。在 `.py` 文件里运行脚本时，`print` 会显示在终端，`return` 不会自动显示。

## 8. 完整示例：华氏度转摄氏度

我们逐步构建一个小例子，把学到的全部内容用上。

```python
def 华氏转摄氏(f):
    """将华氏温度 f 转换为摄氏温度，四舍五入到 1 位小数。"""
    c = (f - 32) * 5 / 9
    return round(c, 1)


def 能否结冰(c):  # c 是摄氏温度
    """判断水在这个温度下是否结冰。"""
    return c <= 0


# ─── 使用 ───
f = 98.6           # 正常体温（华氏）
c = 华氏转摄氏(f)
print(f"{f}°F = {c}°C")        # 98.6°F = 37.0°C
print(f"会结冰? {能否结冰(c)}")  # 会结冰? False

# 测试（用 assert 一次性验证很多情况）
assert 华氏转摄氏(32) == 0.0       # 冰点
assert 华氏转摄氏(212) == 100.0    # 沸点
assert 华氏转摄氏(-40) == -40.0    # 有趣：-40°F = -40°C
assert 能否结冰(0) == True
assert 能否结冰(-5) == True
assert 能否结冰(5) == False

print("✅ 全部测试通过")
```

**从这段代码学到的**：
- `def` 定义函数，`return` 返回计算结果
- `round(c, 1)` 是内置函数，四舍五入到 1 位小数
- `f"{f}°F = {c}°C"` 是 f-string，把变量值嵌入字符串
- `assert` 批量验证正确性
- 函数可以调用另一个函数（`华氏转摄氏` 被 `能否结冰` 间接依赖）

## 9. 例题

### 例 1：求三个数的平均值

**问题**：写一个函数 `mean3(a, b, c)` 返回三个数的平均值。

**解**：
```python
def mean3(a, b, c):
    return (a + b + c) / 3

assert mean3(1, 2, 3) == 2.0
assert mean3(0, 0, 0) == 0.0
assert mean3(-1, 0, 1) == 0.0
```

> **常见错误**：写成 `a + b + c / 3` —— 除法的优先级高于加法，这等价于 `a + b + (c/3)`，结果不对。**括号是你的朋友**。

### 例 2：判断正负数

**问题**：写一个函数 `sign(n)`，n > 0 返回 `1`，n < 0 返回 `-1`，n == 0 返回 `0`。

**解**：
```python
def sign(n):
    if n > 0:
        return 1
    elif n < 0:
        return -1
    else:
        return 0

assert sign(5) == 1
assert sign(-3) == -1
assert sign(0) == 0
```

`if/elif/else` 是条件判断。我们下一章会详细展开——这里先让你看看它的样子。

### 例 3：圆的面积与周长

**问题**：写两个函数，`圆面积(r)` 和 `圆周长(r)`，都接收半径 r。

```python
import math       # 引入数学模块，里面有 pi

def 圆面积(r):
    """假设 r >= 0。"""
    return math.pi * r * r

def 圆周长(r):
    return 2 * math.pi * r

assert abs(圆面积(1) - 3.14159) < 0.001
assert abs(圆周长(1) - 6.28318) < 0.001
```

> `import math` 引入了 Python 内置的数学库。`math.pi` 就是 $\pi$（3.14159...）。`abs(a - b) < 0.001` 是一种比较浮点数的安全方法——下一章我们会了解 `math.isclose`。

### 题 2
下面的代码有什么问题？修正它。
```python
def 平方和(a b)     # 计算 a² + b²
return a * a + b * b
```

<details><summary>点击查看答案</summary>

### 题 5
写两个函数 `c_to_f(c)` 和 `f_to_c(f)`，完成摄氏与华氏的互转，并验证 `f_to_c(c_to_f(x))` 约等于 `x`（对任意合法温度）。公式：$F = \frac{9}{5}C + 32$

<details><summary>点击查看答案</summary>

```python
def c_to_f(c):
    return (9 / 5) * c + 32

def f_to_c(f):
    return (f - 32) * 5 / 9

# 往返验证（round trip test）
for temp in [0, 10, 20, 30, 100, -40]:
    assert abs(f_to_c(c_to_f(temp)) - temp) < 1e-9
```
</details>

## 本章核心概念速查

| 概念 | 一句话 |
|------|--------|
| 表达式 | 可求值的代码片段，如 `2 + 3` |
| 值 | 表达式求值的结果，如 `5` |
| 赋值 | `x = 5` 将值绑定到名字 |
| 名字/变量 | 指向值的标签 |
| `def` | 定义函数的语句 |
| 参数（parameter） | 函数定义时的占位名，如 `def f(x)` 中的 `x` |
| 实参（argument） | 调用时传入的具体值，如 `f(5)` 中的 `5` |
| `return` | 函数把计算结果交给调用者 |
| `None` | 没有返回值时的默认值 |
| `assert` | 条件不成立时报错，用于测试 |
| f-string | `f"文本{变量}"` 将值嵌入字符串 |
| 浮点误差 | 用 `abs(a-b) < 1e-9` 而非 `a == b` 比较浮点数 |
