## Q1 ⭐ 斐波那契（树递归）
实现 `fib(n)` 用树递归（不优化），返回第 n 个斐波那契数（`F(0)=0, F(1)=1`）。

## Q2 ⭐⭐ 组合数
实现 `C(n, k)` 用树递归返回组合数（从 n 选 k）。利用公式 `C(n, k) = C(n-1, k-1) + C(n-1, k)`。

## Q3 ⭐⭐ 网格路径
实现 `grid_paths(m, n)`：m×n 网格，从左上走到右下，每次只向右或向下。返回路径数。

## Q4 ⭐⭐⭐ 硬币找零
实现 `coin_change(amount, coins)`：用 `coins` 列表中面额的硬币凑出 `amount` 的方法数。每种硬币无限量。`coins` 是排序好的元组（从小到大），假设第一个元素是 1。

## Q5 ⭐⭐ 整数划分
实现 `count_partitions(n, m)`：返回 n 划分成**不超过 m** 的部分的方法数。

## Q6 ⭐⭐⭐ 生成子集
实现 `subsets(lst)`：返回列表所有子集的列表。每个子集用 list 表示。返回顺序不作要求。

## Q7 ⭐⭐⭐ 汉诺塔移动序列
实现 `hanoi(n, source, target, auxiliary)`：**返回**移动步骤的字符串列表，如 `["A→C", "A→B", "C→B", ...]`。source/target/auxiliary 是字符串。

## Q8 ⭐⭐ 上楼梯
实现 `triple_steps(n)`：每次可以走 1、2 或 3 级台阶。返回走到第 n 级的方法数。

## Q9 ⭐⭐ 帕斯卡三角形第 k 行
实现 `pascal_row(k)`：返回帕斯卡三角形（杨辉三角）第 k 行的列表，0-indexed。用树递归（每次求前一行）。

## Q10 ⭐⭐⭐ 生成排列
实现 `permutations(lst)`：返回列表所有全排列的列表。每个排列用 list 表示。返回顺序不作要求。

## Q11 ⭐⭐ 最大子集和
实现 `max_subset_sum(lst)`：返回**非相邻元素**的最大和。即选出的元素在列表中不能相邻。用树递归（包含/不包含当前元素的两个分支）。

## Q12 ⭐⭐⭐ 找零方案（列出方案）
实现 `coin_change_ways(amount, coins)`：返回所有找零方案的列表（不是计数）。每个方案是一个列表，如 `[1,1,1,1]` 表示 4 个 1 分。方案中硬币顺序不重要，但每个方案不应重复。
