# Python基础 · 第五章 · 递归 — 作业

> 完成 `homework.py`。运行 `python3 grader.py` 自动批改。
> **所有题目必须用递归实现**，不允许使用 `for`/`while` 循环（除非题目明确允许）。

## Q1 ⭐ 递归阶乘
实现 `factorial(n)` 用递归。n ≥ 0，返回 n!。

## Q2 ⭐ 各位数字之和
实现 `sum_digits(n)` 用递归。返回正整数 n 的各位数字之和。

## Q3 ⭐⭐ 递归求幂
实现 `power(x, n)` 用递归。返回 x^n（n ≥ 0，整数）。不允许用 `**` 或 `pow`。

## Q4 ⭐⭐ 判断回文
实现 `is_palindrome(s)` 用递归。判断字符串 s 是否正反读相同。忽略字母大小写（全部转为小写比较）。

## Q5 ⭐⭐ 递归计数
实现 `count_up(n)` 用递归。**打印**从 1 到 n 的每个数字（每行一个），不可以使用循环。

## Q6 ⭐⭐⭐ 递归最大公约数
实现 `gcd(a, b)` 用递归（递归欧几里得算法）。

## Q7 ⭐⭐ 列表长度
实现 `my_len(lst)` 用递归返回列表长度。不允许用 `len`。

## Q8 ⭐⭐⭐ 反转字符串
实现 `reverse_string(s)` 用递归返回反转后的新字符串。不允许用 `s[::-1]` 或 `reversed()`。

## Q9 ⭐⭐ 递归求和
实现 `recursive_sum(lst)` 用递归返回列表中所有元素的和。

## Q10 ⭐⭐⭐ 递归最大值
实现 `recursive_max(lst)` 用递归返回列表中最大的元素。列表至少有一个元素。不允许用 `max`。

## Q11 ⭐⭐ 统计出现次数
实现 `count(x, lst)` 用递归返回 `x` 在 `lst` 中出现的次数。

## Q12 ⭐⭐⭐ 二分查找
实现 `binary_search(lst, target)` 用递归在**已排序**列表 `lst` 中查找 `target`。若找到返回索引，若找不到返回 -1。需要额外的辅助函数处理 lo/hi 参数。
